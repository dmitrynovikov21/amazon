VPS Startup Commands

1. Start Chrome with CDP

2. Start Server
cd C:\Users\green.eldar\parser2\amazon-main\parser && node server.js
3. Start Worker
cd C:\Users\green.eldar\parser2\amazon-main\parser && node worker.js
4. Update from GitHub
cd C:\Users\green.eldar\parser2\amazon-main && git pull origin main && cd parser && npm install
